## ğŸ“¦ Installation

1.  Open **Visual Studio Code**.
2.  Go to the **Extensions** view (`Ctrl+Shift+X` or `Cmd+Shift+X`).
3.  Search for `Bubble Tea Theme`.
4.  Click **Install**.
5.  Go to **File > Preferences > Theme > Color Theme** and select your preferred flavor: **Bubble Tea Theme** or **Bubble Tea Classic**.
6.  (Optional) Go to **File > Preferences > Theme > File Icon Theme** and select **Bubble Tea Icons**.
